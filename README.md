## Main Feature
- **Eksplorasi Kegiatan:** Cari dan filter berbagai program relawan berdasarkan kategori, lokasi, dan tanggal.
- **Pendaftaran Relawan:** Proses pendaftaran yang mudah dan cepat langsung, mengetahui progress status pendaftaran.
- **Manajemen Organisasi:** Memungkinkan institute untuk memposting kegiatan dan mengelola penerimaan volunteer.
- **Dashboard User:** Memantau status pendaftaran dan riwayat kegiatan yang diikuti.
